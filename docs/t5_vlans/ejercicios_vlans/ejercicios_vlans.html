Solución al ejercicio de VLANs 1

Configurar un switch Cisco mediante la línea de comandos cumpliendo los requisitos siguientes:

  • Mostrar las VLANs

  • Poner el switch en modo servidor dentro del dominio aulab14.com y poniendo la clave admin1234vtp

  • Crear estas VLANs con exactamente estos identificadores y estos nombres:USUARIOS01 con ID 100, GESTION con ID 110, ANALISIS con ID 130, CLIENTES con ID 60, USUARIOS03 con ID 170, TECNICOS con ID 180, DISENO con ID 20

  • Poner estas interfaces en modo acceso asignándolas a estas VLAN:interfaz FastEthernet 0/7 a VLAN 80,interfaz FastEthernet 0/8 a VLAN 50,interfaz FastEthernet 0/14 a VLAN 150,interfaz FastEthernet 0/9 a VLAN 90.Poner estas interfaces en modo troncal asignándolas a estas VLAN:interfaz FastEthernet 0/1 a VLAN 160,interfaz FastEthernet 0/6 a VLAN 70

La solución detallada sería esta:

Switch1>enable
Switch1#configure terminal
Switch1(config)#show vlan
Switch1(config)#vtp domain aulab14.com
Switch1(config)#vtp password admin1234vtp
Switch1(config)#vtp mode server
Switch1(config)#vlan 100
Switch1(config-vlan)#name USUARIOS01
Switch1(config-vlan)#vlan 110
Switch1(config-vlan)#name GESTION
Switch1(config-vlan)#vlan 130
Switch1(config-vlan)#name ANALISIS
Switch1(config-vlan)#vlan 60
Switch1(config-vlan)#name CLIENTES
Switch1(config-vlan)#vlan 170
Switch1(config-vlan)#name USUARIOS03
Switch1(config-vlan)#vlan 180
Switch1(config-vlan)#name TECNICOS
Switch1(config-vlan)#vlan 20
Switch1(config-vlan)#name DISENO
Switch1(config-vlan)#interface FastEthernet 0/7
Switch1(config-if)#switchport mode access
Switch1(config-if)#switchport access vlan 80
Switch1(config-if)#exit
Switch1(config)#interface FastEthernet 0/8
Switch1(config-if)#switchport mode access
Switch1(config-if)#switchport access vlan 50
Switch1(config-if)#exit
Switch1(config)#interface FastEthernet 0/14
Switch1(config-if)#switchport mode access
Switch1(config-if)#switchport access vlan 150
Switch1(config-if)#exit
Switch1(config)#interface FastEthernet 0/9
Switch1(config-if)#switchport mode access
Switch1(config-if)#switchport access vlan 90
Switch1(config-if)#exit
Switch1(config)#interface FastEthernet 0/1
Switch1(config-if)#switchport mode trunk
Switch1(config-if)#switchport trunk allowed vlan 160
Switch1(config-if)#exit
Switch1(config)#interface FastEthernet 0/6
Switch1(config-if)#switchport mode trunk
Switch1(config-if)#switchport trunk allowed vlan 70
Switch1(config-if)#exit
